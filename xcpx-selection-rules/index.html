XCPX省赛选拔赛规则

发表于 2026-03-12 更新于 2026-03-13 分类于 比赛通知
XCPX 省赛选拔赛的比赛平台、赛制安排、纪律要求、工具限制与积分排名规则说明

XCPX省赛选拔赛规则

1. 比赛平台与赛制

  • 比赛将在 Codeforces 和 Virtual Judge (VJ) 两个平台同时进行。
  • 每次选拔赛由两部分合并而成：一场 Codeforces 的 Div3 比赛 + VJ 上的一场 AtCoder Beginner Contest (ABC)。
  • 两个平台的比赛同时开启并进行，总时长共计 3 小时。
  • 每场比赛的最终结果将由选手在两个平台上的成绩合并计算并进行单场排名。

2. 比赛纪律与防作弊

  • 比赛过程要求全程有声录屏。
  • 任何作弊行为一经发现，将直接取消选拔资格。

3. 比赛时间与请假制度

  • 计划共举行 5 场选拔赛，常规比赛时间为每周五晚上 18:30–21:30。具体排期如下：
    • 第一场：3月27日 18:30–21:30
    • 第二场：4月3日 18:30–21:30
    • 第三场：4月10日 18:30–21:30
    • 第四场：4月17日 18:30–21:30
    • 第五场：4月24日 18:30–21:30
  • 注：若某周遇到特殊情况（如节假日或学校安排）无法按原计划进行，将会更改比赛时间并提前通知。
  • 若因故不能参加某场比赛，必须提前告知负责人。

4. 比赛题目

  • 题目来源于过往的 CF Div3 与 AtCoder ABC 场次。
  • 比赛期间严禁通过任何方式查找题目来源及题解。

5. 允许与禁止使用的工具

  • 禁止使用：任何 AI 模型、搜索引擎。
  • 允许使用：翻译网站/软件、本地编辑器（IDE）、自己整理的代码板子（允许复制粘贴）。

6. 选拔与积分排名规则

  • 排名计算：5 场比赛取每人单场成绩（排名数值）最好的 3 场进行加总排序（根据 \(n/2+1\) 规则，5场选3场）。最终总分数值越小，总排名越靠前。
    • 例如：某选手 5 场的单场排名分别为 1、4、2、3、2，则取成绩最好的 3 场（1、2、2），他的最终成绩分为 5。
  • 缺考处理：未参加（缺考）的场次，一律按 0 题通过（0AC）来计算该场的排名。
  • 特殊情况处理：若出现特殊情况导致无法按计划举办完 5 场比赛，则选取每人成绩最好的 x 场进行计算。
    • 计算公式为：x = n/2 + 1（n 为实际举办的总场数，n/2 向下取整）。
    • 对应标准为：5 选 3、4 选 3。
