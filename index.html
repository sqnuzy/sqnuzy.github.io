第1章 前缀与差分类

1. 前缀积

用途：区间乘积查询（模质数，使用逆元）。
复杂度：预处理 \(O(n)\)，单次查询 \(O(\log mod)\)。
注意：要求模数为质数且前缀积非 0 时可逆。

2. 前缀置换

用途：维护前缀交换操作形成的排列，查询区间操作后的最终排列。
复杂度：预处理 \(O(10n)\)，单次查询 \(O(10\log 10)\)。
注意：这里只适合值域固定很小（这里是 10 个球）的情况。

3. 前缀矩阵积

用途：2×2 转移矩阵区间连乘。
复杂度：预处理 \(O(n)\)，单次查询 \(O(1)\) 次矩阵运算。
注意：这里用到了矩阵求逆来实现区间矩阵积： \[ M_{[l,r]} = pre_{l-1}^{-1} \cdot pre_r \]

4. 前缀和 + 等差数列

用途：区间贡献是 \(1,2,3,\dots\) 形式时，用多次前缀和实现。
复杂度：\(O(n)\)。

5. 前缀和 + 平方数列

用途：区间后缀加 \(1,4,9,\dots\) 这类可由多次前缀和生成的序列。
技巧：

  • 常数列：1 次前缀和
  • 等差列：2 次前缀和
  • 平方列：3 次前缀和

6. 前缀和 + 多项式

用途：静态批量区间加多项式，最后统一查询区间和。
核心：高阶差分 + 小次数多项式。
注意：这份代码是竞赛向写法，适合 \(k \le 5\)。

7. 高阶前缀和 / 差分（NTT）

用途：对数组做 \(k\) 次前缀和 / 差分，\(k\) 可正可负且很大。
复杂度：\(O(n\log n)\)。
核心：组合数卷积。

8. SOS DP（子集和变换）

用途：查询某集合所有子集 / 超集的值和。
复杂度：预处理 \(O(n2^n)\)，查询 \(O(n)\) 或 \(O(1)\) 读答案。

第2章 区间数据结构

1. 线段树（区间乘、区间加、区间和、平方和）

用途：同时维护

  • 区间和
  • 区间平方和
  • 区间乘法修改
  • 区间加法修改

复杂度：单次修改 / 查询 \(O(\log n)\)。
注意：懒标记是仿射变换 \(x \mapsto ax+b\)。

2. 树状数组 1：单点加 / 区间和

复杂度：单次修改 / 查询 \(O(\log n)\)。

3. 树状数组 2：区间加 / 单点查

技巧：维护差分数组的前缀和。
复杂度：单次 \(O(\log n)\)。

4. 树状数组求逆序对

复杂度：\(O(n\log n)\)。
注意：若相同值不算逆序对，排序时相同值按下标降序。

5. 树状数组处理二维偏序

用途：把二维条件转成排序 + 一维树状数组。
套路：

  • 一维排序处理一个约束
  • 另一维离散化后进树状数组

第3章 倍增与 RMQ

1. ST 表

用途：静态 RMQ。 复杂度：预处理 \(O(n\log n)\)，查询 \(O(1)\)。 注意：适合静态问题，不支持修改。

2. 倍增表（函数图）

用途：每点恰有一条出边时，求走 \(k\) 步后的信息。 可维护：

  • 第 \(2^j\) 步到哪
  • 权值和
  • 权值最小值

第4章 其他常用结构

1. 三维前缀和

复杂度：预处理 \(O(n^3)\)，单次查询 \(O(1)\)。

2. 字符串哈希（双模）

用途：判断两个子串是否相等。
复杂度：预处理 \(O(n)\)，查询 \(O(1)\)。
注意：双模更稳。

3. 最小表示法

用途：求循环串中字典序最小的表示。
复杂度：\(O(n)\)。

4. rope

用途：大字符串 / 大序列的拼接、插入、删除。
复杂度：一般视作平衡树，单次操作近似 \(O(\log n)\)。
注意：GCC 扩展。

5. ordered_set

用途：支持求排名、求第 \(k\) 小。
注意：不能直接存重复值，若要存重复值可存 pair<值, 编号>。

6. 普通可换根 Trie（统计前缀数量版）

用途：

  • 插入字符串
  • 查询某前缀出现次数
  • 整体加同一前缀字符（换根）
