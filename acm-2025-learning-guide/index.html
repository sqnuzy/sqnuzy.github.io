25级 ACM 学习资料导航

发表于 2026-04-16 分类于 竞赛入门
按入门、训练、工具与比赛四个方向重新整理 25 级学习资料，方便后续在博客中持续维护。

这篇文章是对 25级学习 资料目录的一次重新整理版导航，目标不是把所有文件原样搬上来，而是把真正适合公开阅读、方便后续维护的内容沉淀成博客。

先看什么

如果你刚接触算法竞赛，建议按下面顺序阅读：

  1. 先看 25级第一次线上培训整理，了解训练要求、常用网站和基本规范。
  2. 再看 工具环境与 VJudge 提交问题排查，把写题和提交环境先配好。
  3. 接着看 字符串哈希与 vector 建图入门笔记，补一部分常见实现细节。
  4. 如果准备蓝桥杯，再看 蓝桥杯比赛技巧整理，把比赛中的得分思路先建立起来。
  5. 平时训练时，建议按 错题分析怎么写 的模板记录自己的失误点。

这次整理后的结构

1. 入门说明

  • 第一次线上培训的重点内容已经改写成博客文章。
  • 原目录中的网站链接、训练要求、题解要求也一并合并进去了。

2. 工具与环境

  • 把 ScriptCat 插件教程、Typora 修改字体、VJ 提交不成功解决办法 合并成一篇更完整的工具说明。
  • 原资料里的截图路径是本机临时路径，没法直接用于博客，因此改成了文字化步骤。

3. 学习笔记

  • 字符串哈希、vector建图.md 已经按“场景 -> 原理 -> 模板 -> 易错点”的顺序重写。
  • 蓝桥杯技巧.md 已经整理成更适合赛前复习的版本。
  • zy错题分析.md 没有原样公开，而是提炼成了更通用的错题分析模板与方法。

没有公开同步的内容

下面这些内容不适合放到公开博客，所以这次没有同步：

  • AcWing账号分配.txt
  • AcWing账号密码.txt
  • ACM2025 id统计.xlsx
  • 天梯赛.xlsx
  • 天梯赛选拔赛成绩.xlsx
  • 体积很大的 算法基础课视频/Week*.zip
  • 可能涉及版权问题的整本教材或电子书 PDF

这些资料更适合继续保留在社团内部资料库、本地硬盘或单独网盘中。

后续维护建议

  • 新的培训内容，尽量直接写成 Markdown，而不是只保留 PDF。
  • 图片尽量放到博客可引用的相对路径里，不要继续使用 Typora 的本机临时图片目录。
  • 比赛经验类内容建议统一成“背景、题目、失误点、正确思路、复盘结论”五段式，后续检索会轻松很多。

如果后面还要继续整理别的年级资料，可以直接沿用这套结构。
