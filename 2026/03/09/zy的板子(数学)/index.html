定理：

卢卡斯定理描述了在一个素数模 \(p\) 下二项式系数的性质。其公式如下：

\[ \dbinom{n}{k} \equiv \prod_{i=0}^{m} \dbinom{n_i}{k_i} \pmod{p} \]

其中，整数 \(n\) 和 \(k\) 分别用 \(p\) 进制表示为：

\[ n = n_m p^m + n_{m-1} p^{m-1} + \dots + n_0 \]

\[ k = k_m p^m + k_{m-1} p^{m-1} + \dots + k_0 \]

示例

设 \(p=5\)，计算 $ $。

首先，将 \(7\) 和 \(3\) 转换为 \(5\) 进制：

  • \(7_{10} = 12_5\)
  • \(3_{10} = 03_5\)

根据卢卡斯定理：

\[ \dbinom{7}{3} \equiv \dbinom{1}{0} \times \dbinom{2}{3} \pmod{5} \]

由于 $ = 0$，因此：

\[ \dbinom{7}{3} \equiv 0 \pmod{5} \]

插板法

对于从 \(m\) 个不同元素中可重复地选取 \(n\) 个元素的组合数，其值为：

\[ \binom{m + n - 1}{n} = \binom{m + n - 1}{m - 1} \]

基本问题模型

  • 问题： 将 n 个完全相同的物品，分配给 k 个不同的对象（或分成 k 个非空的组），有多少种不同的分配方案？

  • 答案: \(C_{n - 1}^{k - 1}\)​

变体 1：允许对象为空（允许空组）

  • 问题： 将 n 个完全相同的物品，分配给 k 个不同的对象，允许某些对象分得 0 个物品（即允许空组），有多少种分配方案？

  • 答案: \(C_{n + k - 1}^{k - 1}\)

变体 2：要求每个对象至少分得 m 个 (m >= 1)

  • 问题： 将 n 个完全相同的物品，分配给 k 个不同的对象，要求每个对象至少分得 m 个 (m >= 1)，有多少种分配方案？

  • 答案: \(C_{n - k * m + k - 1}^{k - 1}\)
变体 3：要求每个对象至少分得 mᵢ 个 (不同对象要求不同)

  • 问题： 将 n 个完全相同的物品，分配给 k 个不同的对象，要求第 i 个对象至少分得 mᵢ 个 (mᵢ >= 0)，有多少种分配方案？

  • ( \(S=\sum\limits_{i=1}^n\)mᵢ)

  • 答案: \(C_{n - S + k - 1}^{k - 1}\)

多重集合的排列

即考虑重复元素的排列，比如单词 “BANANA” 的不同排列数。
这种情况下，多重排列数的公式是：

\[ \frac{n!}{n_1!\,n_2!\cdots n_k!} \]

其中

  • \(n\) 是总元素个数
  • \(n_i\) 是第 \(i\) 种元素的重复次数

这个公式可以看作是组合数的乘积形式，因为：

\[ \frac{n!}{n_1!\,n_2!\cdots n_k!} = \binom{n}{n_1} \cdot \binom{n - n_1}{n_2} \cdot \binom{n - n_1 - n_2}{n_3} \cdots \binom{n_k}{n_k} \]

分拆数

定义

将 \(n\) 分成恰有 \(k\) 个部分的分拆，称为 \(k\) 部分拆数，记作 \[ p(n,k). \]

对应方程

显然，\(p(n,k)\) 也是下面方程的解数： \[ n - k = y_1 + y_2 + \dots + y_k \quad (y_1 \geq y_2 \geq \dots \geq y_k \geq 0). \]

按非零部分计数

若该方程恰有 \(j\) 个部分非零，则恰有 \(p(n - k,\,j)\) 个解，于是 \[ p(n,k) = \sum_{j=1}^{k} p(n - k,\,j). \]

递推关系

对上式相邻两项作差，得 \[ p(n,k) = p(n - 1,\,k - 1) + p(n - k,\,k). \]

范德蒙德卷积

范德蒙德卷积公式

\[ \sum_{i=0}^{k} \binom{n}{i} \binom{m}{k-i} = \binom{n+m}{k} \]

推论1：

\[ \sum_{i=-r}^{s} \binom{n}{r+i} \binom{m}{s-i} = \binom{n+m}{r+s} \]

推论2：

\[ \sum_{i=1}^{n} \binom{n}{i} \binom{n}{i-1} = \binom{2n}{n-1} \]

推论3：

\[ \sum_{i=0}^{n} \binom{n}{i}^2 = \binom{2n}{n} \]

推论4：

\[ \sum_{i=0}^{m} \binom{n}{i} \binom{m}{i} = \binom{n+m}{m} \]

指数生成函数

  1. \(\displaystyle e^x=\sum_{n=0}^{\infty}\frac{x^n}{n!}\)
  2. \(\displaystyle e^{-x}=\sum_{n=0}^{\infty}\frac{(-1)^n x^n}{n!}\)
  3. \(\displaystyle \frac{e^x+e^{-x}}{2}=\sum_{n=0}^{\infty}\frac{x^{2n}}{(2n)!}\)
  4. \(\displaystyle \frac{e^x-e^{-x}}{2}=\sum_{n=0}^{\infty}\frac{x^{2n+1}}{(2n+1)!}\)
