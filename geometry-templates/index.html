1. 点在线段上

这是非常常见的判断。

判断逻辑：

  • 先用叉积判断是否共线

  • 点在顶点上
  • 点在边上
  • 点在内部
  • 点在外部

三、面积公式

1. 三角形面积

三点 A, B, C 构成三角形面积公式：

\[ S = \frac{1}{2} |\overrightarrow{AB} \times \overrightarrow{AC}| \]

2. 多边形面积

如果多边形顶点按顺时针或逆时针顺序给出，就可以用鞋带公式：

\[ S = \frac{1}{2}\left|\sum_{i=1}^{n}(x_i y_{i+1} - x_{i+1} y_i)\right| \]

其中当 i=n 时，i+1 回到 1。

五、旋转卡壳

1. 求凸包直径

旋转卡壳常用来求凸包上最远点对，也就是凸包直径。

思路一般是：

  1. 先求凸包
  2. 再在凸包上跑双指针

注意

  • 这份板子输出的是最远距离的平方
  • 很多题会直接要求输出平方，这样可以减少精度误差

八、圆的基础内容

1. 圆的基本定义

在计算几何里，圆一般记录两个量：

  • 圆心 c
  • 半径 r

说明

这一部分最好单独保留，因为后面“点和圆关系”“直线和圆关系”“圆和圆关系”都会用到。

2. 点和圆的位置关系

设点为 p，圆为 O(c, r)。

判断方法很直接：

  • |p-c| < r：点在圆内
  • |p-c| = r：点在圆上
  • |p-c| > r：点在圆外

说明

这是最基础的圆关系判断，很多题都会先从这里开始。

3. 点在直线上的投影

圆和直线求交时，常常要先求圆心到直线的投影点，所以这个函数建议和圆放在一起。

设点为 p，直线经过 a, b。

说明

这个点就是 p 在直线 ab 上的垂足。 后面“直线和圆交点”会直接用到。

4. 点到直线距离

设点为 p，直线经过 a, b。

公式是：

\[ \frac{|(b-a)\times(p-a)|}{|b-a|} \]

说明

圆和直线的位置关系，本质上就是比较：

  • 圆心到直线距离
  • 圆半径

谁大谁小。

5. 直线和圆的位置关系

设直线经过 a, b，圆为 O(c, r)。

分类如下：

  • 距离 > r：相离
  • 距离 = r：相切
  • 距离 < r：相交

说明

这部分是“直线与圆交点”的前置判断。 先判关系，再决定有没有交点、几个交点。

6. 圆和圆的位置关系

设两个圆分别为 A(c1, r1)、B(c2, r2)，圆心距为 d。

分类：

  • d > r1 + r2：外离
  • d = r1 + r2：外切
  • |r1-r2| < d < r1+r2：相交
  • d = |r1-r2|：内切
  • d < |r1-r2|：内含
  • d = 0 且 r1 = r2：重合

说明

这个函数是圆题最常见的关系判断之一。 尤其是求交点之前，通常先调它判断一下是否合法。

7. 直线与圆的交点

思路

设直线为 ab，圆为 O(c, r)。

做法：

  1. 先求圆心 c 在直线 ab 上的投影点 q
  2. 计算 q 到圆心的距离 d
  3. 比较 d 和 r
  4. 若有两个交点，则从 q 沿直线方向左右各走一段

返回值

  • 0：无交点
  • 1：一个交点
  • 2：两个交点

说明

这是非常常用的板子。 只要你前面把 Projection 和 Distance_to_line 写好了，这里会很好记。

8. 圆与圆的交点

思路

设两个圆为 A(c1, r1)、B(c2, r2)。

步骤：

  1. 先判断两圆是否有交点
  2. 设圆心距为 d
  3. 求出公共弦中点 mid
  4. 沿着垂直方向展开，得到两个交点

返回值

  • -1：无穷多个交点，两个圆重合
  • 0：无交点
  • 1：一个交点
  • 2：两个交点

说明

这里的

是为了防精度误差。 理论上切圆时 h2 = 0，但浮点数有时会算成一个很小的负数。
