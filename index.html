树的直径：

定义：树上任意两点间距离的最大值称为树的直径。通常用 \(D\) 表示长度，\((U, V)\)​ 表示端点。

方法一：两次 DFS
  • 优点：不仅能求出长度，还能求出具体的路径端点（\(U\) 和 \(V\)）。
  • 缺点：只适用于非负权边（如果边权有负数，这个贪心策略会失效）。
方法二：树形 DP 法

这种方法通过一次 DFS 遍历整棵树，在回溯的过程中计算。

  • 优点：
    1. 适用于 负权边。
    2. 只需要 一次 DFS。
    3. 容易嵌入到其他的树形 DP 逻辑中。
  • 缺点：
    • 如果不额外记录，很难直接知道直径的两个端点是谁，只能求出 长度。
1. 最远点性质（解题核心）

结论： 对于树上 任意节点 \(u\)，距离 \(u\) 最远 的节点 \(v\)，一定是树的某一条直径的 一个端点。

应用与推论：

  • 求直径（两次 BFS/DFS 法）：
    1. 任选一点 \(x\)，找到距离 \(x\) 最远的点 \(U\)（\(U\) 必为直径一端）。
    2. 从 \(U\) 出发，找到距离 \(U\) 最远的点 \(V\)（\(V\) 必为直径另一端）。
    3. \(U \to V\) 即为直径。
  • 求任意点的最远距离：
    • 只需要求出直径的两个端点 \(U, V\)。
    • 树上任意点 \(x\) 到树上其他点的最大距离 \(= \max(dist(x, U), dist(x, V))\)。
2. 树的中心 (Tree Center)

定义： 树上所有节点中，到树中其他节点 最大距离最小 的节点。

性质： - 树的中心一定位于 树的直径的中点。 - 一棵树最多有 1 个（直径为偶数）或 2 个（直径为奇数）中心。

求法算法：

  1. 先求出直径的两个端点 \(U, V\) 以及直径长度 \(len\)。
  2. 从 \(U\) 开始 BFS/DFS，记录每个节点的父节点 pre[]（用于还原路径）。
  3. 从 \(V\) 开始，通过 pre[] 往回跳 \(\lfloor len/2 \rfloor\) 步。
  4. 当前所在的节点即为树的中心。
3. 多条直径的交集性质

结论： 如果一棵树有多条直径，它们 一定相交，且所有直径的 交集部分 是一段连续的路径（或一个点）。

扩展应用（求所有直径端点）： - 如果题目要求统计有多少对点能构成直径，或者判断某条边是否在直径上。 - 思路： 1. 求出直径长度 \(L\)。 2. 从任意一个直径端点 \(U\) 出发遍历整棵树。 3. 所有满足 \(dist(U, x) = L\) 的点 \(x\)，都是直径的另一端点。 4. 反过来，从所有这些 \(x\) 出发，距离为 \(L\) 的点也是端点。

4. 树的合并与直径最小化

问题描述： 有两棵树 \(T_A\)（直径 \(D_A\)）和 \(T_B\)（直径 \(D_B\)），在两树之间加一条边，使得合并后的新树直径 最小。

核心思想： - 贪心策略：为了让新直径最小，应该把两棵树的 中心 连起来。 - 树的半径 \(R = \lceil D / 2 \rceil\)（即中心到最远点的距离）。

新直径公式： \[ D_{new} = \max \left( D_A, \quad D_B, \quad \lceil \frac{D_A}{2} \rceil + \lceil \frac{D_B}{2} \rceil + 1 \right) \] > 注：公式中的 \(+1\) 是因为连接两棵树的那条新边贡献了 1 的长度。

分拆数

定义

将 \(n\) 分成恰有 \(k\) 个部分的分拆，称为 \(k\) 部分拆数，记作 \[ p(n,k). \]

对应方程

显然，\(p(n,k)\) 也是下面方程的解数： \[ n - k = y_1 + y_2 + \dots + y_k \quad (y_1 \geq y_2 \geq \dots \geq y_k \geq 0). \]

按非零部分计数

若该方程恰有 \(j\) 个部分非零，则恰有 \(p(n - k,\,j)\) 个解，于是 \[ p(n,k) = \sum_{j=1}^{k} p(n - k,\,j). \]

递推关系

对上式相邻两项作差，得 \[ p(n,k) = p(n - 1,\,k - 1) + p(n - k,\,k). \]

多边形面积

只要顶点的坐标是按顺序（顺时针或逆时针均可）排列的，多边形的面积等于所有相邻顶点向量叉积之和的一半。

\[ S = \frac{1}{2} \left| \sum_{i=1}^{n} (x_i y_{i+1} - x_{i+1} y_i) \right| \]

注意：当 \(i=n\)（最后一个点）时，\(i+1\) 需要回到第一个点（即 \(x_{n+1} = x_1, y_{n+1} = y_1\)）。
